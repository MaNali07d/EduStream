# 🧠 Project Overview

EduStream is a **modern EdTech dashboard platform** designed to simulate an **AI-powered learning ecosystem**.

It enables **students, teachers, and admins** to interact within a unified learning environment that supports:

- course management
- quizzes
- assignments
- AI study assistant
- automated quiz generation
- progress analytics

All AI features are **simulated using rule-based backend logic** so the system runs **fully offline without external APIs**,

# ✨ Core Features

## 🔐 Authentication System

✔ Login / Signup  
✔ Role Selection (Student / Teacher)  
✔ Secure session handling  
✔ User data stored locally

---

# 📚 Course Management

👨‍🏫 Teachers can:

- Create courses
- Upload lecture videos
- Add course materials
- Manage enrollments

👨‍🎓 Students can:

- Browse courses
- Enroll in courses
- Watch lectures

Supported video formats:

▶ YouTube Embed  
▶ Local Video Player

---

# 📝 Assignment System

Teachers can:

✔ Create assignments  
✔ Set deadlines  
✔ Review submissions  

Students can:

✔ Submit text answers  
✔ Upload before deadline  
✔ Receive automated feedback  

---

# 🧪 Quiz System

Teachers can:

- create quizzes
- add MCQ questions
- define correct answers

Students can:

- attempt quizzes
- get instant results
- review correct answers

Quiz results automatically appear on the **student dashboard**.

---

## 📖 Study Assistant

Students can ask questions like: Explain neural networks, Explain photosynthesis
The system searches a **knowledge dataset** and returns explanations.
If the topic is not found: "This topic will be covered in upcoming lessons.


---


# ⚙ AI Quiz Generator

Teachers paste study notes.
Example input: Photosynthesis is the process by which plants make food.
Generated question: What is photosynthesis?


The system automatically generates **3–5 quiz questions**.

---

# 🧠 Assignment Feedback AI

After submission the system checks:

✔ Grammar issues  
✔ Sentence clarity  
✔ Text length  
✔ Repeated words  

Example feedback:
Your answer is detailed.
Try improving sentence clarity.
Add more examples.


---

# 📊 Student Progress Dashboard

Displays:

📈 Course completion  
📊 Quiz scores  
📝 Assignment completion  

Charts are generated using **Chart.js**.

---

# 🗺 Learning Path Generator

- subject
